O que acontece ao iniciar um repositório? Por que você precisa fazer isso?

	É criado uma subpasta de nome .git
	Esta é a pasta de armazenamento dos metadados para o controle de versão


Como você pode usar a área de preparação para garantir que tenha um commit por alteração lógica?

	Você pode adicionar a alteração lógica através do comando: git add <nome do arquivo> e realizar
	o commit


Quais são algumas das situações em que branches seriam úteis para manter seu histórico organizado? Como branches ajudariam?

	Eles permitem experimentar novos caminhos de desenvolvimento ou criar versões específicas mantendo uma base em comum estável

Como os diagramas ajudam a visualizar a estrutura de branches?

	Permitem encontrar rapidamente em qual momento temporal foram criados em relação a estrutura raiz
